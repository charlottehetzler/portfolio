Projects.

Mister Cocktail

Create and share your cocktail recipes.

This is a simple website to store cocktail recipes. Each cocktail is displayed on a page showing its ingredients and a picture that can be uploaded.
You can edit your cocktail recipe or delete it from the database. This website was made for one user and is meant to be personal and engaging.
The titles are more personal than a usual website's would be.

Sockciety

Find your missing sock at Sockciety.

Sockciety was my first fun group project - a platform providing a solution to the omnipresent problem of loosing socks.
Users can scroll trough all exisiting socks and search for a specific one based on various filters. When having a closer look on a single sock the user is provided with information like description, price, location and reviews about the owner.

Joyage

Find cheap flights for your city trip with friends.

Final project at Le Wagon.
Joyage finds the cheapest flight for a group of friends to meet at a common destination.
It provides different possible destinations based on the user's different starting points and their budget. The application is build by using the Amadeus API.
In this way, planning a group trip and booking flights can be done within a few minutes!

Onlineshop

Myos is a simple online shop where users can view all products and search for a specific one via a search interface.
Users can have a closer look on each product before adding it to the shopping cart which saves its items until it is transformed into an order by entering the email adress.
Wihtin the shopping cart the user is able to change the amount of each product or even delete it completely from the cart.
