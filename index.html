Hi, I am Charly

After joining the Le Wagon bootcamp in Tel Aviv last year and a couple of months freelancing, I started working as a Junior Software Engingeer in Berlin.

Joyage

My final project at Le Wagon. Joyage finds the cheapest flight for a group of friends to meet at a common destination. It provides possible destinations based on the user's different starting points and their budget. The application is build by using the Amadeus API. In this way, planning a group trip and booking flights can be done within only a few minutes!

Myos Onlineshop

Myos is a simple online shop where users can view all products and search for a specific one via a search interface. Users can have a closer look on each product before adding it to the shopping cart that saves its items until it is transformed into an order by entering the email adress. Wihtin the shopping cart the user is able to change the amount of each product or even delete it completely from the cart.

Sockciety

Sockciety was my first fun group project - a platform providing a solution to the omnipresent problem of loosing socks. Users can scroll through all exisiting socks and search for a specific one based on various filters. When having a closer look on a single sock the user is provided with information like description, price, location and reviews about the owner.

Strand Parken

The customer wanted an app that his employees could use to create and print parking tickets. Via a platform the admin has access to all issued tickets, which he can export into an Excel file. In addition, he can create, edit or delete ticket-relevant information such as price and parking space.
